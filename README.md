# Club-do-Terror


Bem-vindo ao **Club do Terror**!  
Este projeto reúne histórias, imagens e códigos relacionados ao gênero de terror.

## Estrutura do projeto
- `src/` → código-fonte principal
- `README.md` → documentação inicial

## Objetivo
Criar um espaço para compartilhar e desenvolver conteúdos de terror, seja em forma de narrativa, imagens ou aplicações.
